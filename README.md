#  Easy Do - Todo App

A modern, user-friendly Todo application built using **HTML, CSS, and JavaScript** with support for multiple users, task management, and real-time API integrations.

##  Features

*  User Authentication (Register & Login)
*  Multi-user support (separate todos per user)
*  Add, Edit, Delete Todos
*  Mark tasks as Completed / Pending
*  Filter Todos (All / Completed / Pending)
*  Add Date and ⏰ Time to tasks
*  Dynamic Quote API integration
*  Weather API integration
*  Data persistence using LocalStorage
*  Clean and responsive UI


##  Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API (Async/Await)
* LocalStorage

##  How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/todo-app.git
```

2. Open the project folder:

```
cd todo-app
```

3. Open `ToDo-Index.html` in your browser
